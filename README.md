# 2048

Классическая головоломка **2048** с авторежимом и рейтингом.  
Чистый HTML/CSS/JS — без сборки и без сервера. Работает на GitHub Pages и офлайн.

## Возможности

- **Размеры поля**: 2×2 … 8×8 (мини → эпос)
- **Особые режимы**: Четвёрки, Хаос, Джокер, Спринт (50 ходов), Блиц (60 сек), Широкий хаос
- У каждого режима свой рекорд и цель
- Управление: стрелки / WASD / свайп / кнопки на телефоне
- **Авторежим** — ИИ сам выбирает ходы (эвристика: пустые клетки, монотонность, гладкость)
- **Рейтинг** — топ результатов на устройстве (localStorage), с фильтром по режиму

## Как открыть локально

Открой `index.html` через любой статический сервер (модули ES требуют http/https):

```bash
# Python
python3 -m http.server 8080

# или Node
npx --yes serve .
```

Затем зайди на http://localhost:8080

## Публикация на GitHub Pages

После мержа в `main` сработает workflow **Deploy to GitHub Pages**.

Один раз в настройках репозитория:

1. **Settings → Pages**
2. Source: **GitHub Actions**

Игра будет доступна по адресу:

`https://bfrees23.github.io/dwadwago/`

## Структура

```
index.html          # страница
css/styles.css      # оформление
js/modes.js         # режимы поля
js/game.js          # логика 2048
js/ai.js            # авторежим
js/leaderboard.js   # рейтинг (localStorage)
js/app.js           # UI и управление
```

## Рейтинг

Результаты хранятся в браузере на устройстве игрока. Так игра остаётся полностью статической и всегда работает без внешних API. Глобальный онлайн-рейтинг потребовал бы отдельный бэкенд.
