Рейтинг

Общий рейтинг хранится в репозитории (`data/leaderboard.json`) и обновляется через GitHub Issues + Actions.

    Пока пусто — сыграй и отправь результат.

    Как играть

    • Сверху над полем выбери режим — в списке или кнопками (размеры 2×2–8×8 и особые правила).
    • Свайпай или жми стрелки — плитки скользят в выбранную сторону.
    • Отмена (кнопка или Ctrl/Cmd+Z) откатывает ход.
    • Авторежим — улучшенный ИИ (expectimax + snake-эвристика) в Web Worker; можно выбрать скорость и силу.
    • Локальный рейтинг — только на этом устройстве.
    • Рейтинг GitHub: кнопка «В GitHub» откроет Issue с результатом → Action запишет его в data/leaderboard.json → сайт подтянет общий топ.

    Вход админа

    Войти может только владелец репозитория через свой GitHub Personal Access Token. Чужой аккаунт будет отклонён.
